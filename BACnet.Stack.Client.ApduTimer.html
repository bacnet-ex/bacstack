    BACnet.Stack.Client.ApduTimer (bacstack v0.1.0)

    Copy Markdown View Source

    Internal module for BACnet.Stack.Client.

    It is used as APDU timer for outgoing APDUs. It holds together all the necessary information to track the APDU, time, retry count and contains information that is used to reply to the application.
